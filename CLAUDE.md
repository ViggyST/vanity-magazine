## 1. North Star / Scope

A private, laptop-first personal vault: Projects showcase + mixed-length Blog + Learning roadmap
(Kanban) + Now page. NOT a public portfolio — habit-first, built for personal use, unlisted URL,
no auth/password gate. Repo: `ViggyST/vanity-magazine`.

**In scope (MVP):** final 15-project roster, Vercel deploy, Supabase-backed blog/learning/now
tables, password-free admin entry form, `/blog` `/learning` `/now` pages, thumbnails.

**Out of scope:** mobile/PWA optimization, full CRUD admin UI (table editor covers manual edits),
auth beyond unlisted URL, Astro rebuild, visual redesign (deferred to post-Session-6 usage period),
public sharing/SEO.

## 3. Conventions

- Design tokens: near-black bg, off-white text, muted gold accent `#D6C7A1`. Fraunces (display) +
  Inter (body). Typography-first, restrained motion (`--transition-fast/base/slow`).
- `category` field (legacy) drives `getCoverForCategory()` cover-image fallback — do NOT repurpose
  or delete it when adding `typeBucket`. New projects without a real cover intentionally fall back
  to `coverFallback` via an unmapped `category` string.
- Project `id`s are stable strings, referenced by `seedPosts[].projectId` — never renumber existing
  ids when editing the roster.
- TBD — date/time handling helper: define during Session 1 if new date logic is added, else n/a
  (seed data only, no live date logic yet).

## 8. Tool Division

> Claude Code (desktop app or CLI) requires a **local clone** of this repo — it edits files on
> disk directly. It has no access to a repo that only exists on GitHub. If not cloned yet:
> `git clone https://github.com/ViggyST/vanity-magazine.git <local-folder>`, then open Claude Code
> Desktop pointed at that folder.

| | Claude Code | Cursor |
|---|---|---|
| Use for | Anything matching a numbered Build Session — multi-file builds, new features, schema changes | Git operations, running the dev server, single-file/CSS tweaks, debugging runtime errors |
| Default if unsure | Touches >1 file or adds new logic → **Claude Code** | One-line fix, no behavior change → **Cursor** |
| Requires | Local repo clone (writes to filesystem) | Local repo clone (same reason) |
